Projects

selfLESS

View GitHub / View Demo

This project started as a clone of the Airbnb landing page. While I was building it, I began to imagine what an extreme endpoint of the gig economy would look like. So I pivoted. The goal of the page now is to attract what selfLESS calls Providers, a gig worker who rents out their time.

Featured on the page is a clickable icon that switches between a light theme and a dark theme for those graveyard shift Providers. There is also a calculator that calculates earnings given type of labor and length of rental.

Kimbap Super

View GitHub / View Demo

I wanted to learn what went into making a website reponsive, so I built a one-page webpage for a fictional Korean restaurant. The goal of the page is to provide all the relevant information a restaurant guest might need.

The page features the most eye-catching images I could source from Unsplash, a menu rendered with the help of a JS script, and contact information. The parallax effect is disabled on mobile.

About

My first experience with programming was a game called RPG Maker 2 for the Playstation 2. Working in the game was a clunky, frustrating, and time-consuming process; it involved navigating endless menus because a gamepad is not a keyboard. But I stuck with it - learning how to create scripts; work with variables and conditions; and design visual effects - and the results were not that impressive. But it was an extremely rewarding feeling when everything worked the way I wanted it to.

It's this feeling of creating something functional that made me decide to pivot into development. I began my career path as an ESL teacher in South Korea, before working professionally as a writer back in the U.S. Since delving into web development, I have gained proficiency in HTML, CSS, and JS, plus a familiarity with back-end technologies like NodeJS. Now, as a self-taught developer, I am looking for opportunities to apply my skills and creativity to solve technical problems in the real world.
